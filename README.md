# BOKUZUSHI - 星砕き

Three.js で作ったブロック崩しゲーム。ブロックを壊して段位を上げ、強い色で貫通して星を砕け。

## 遊ぶ

https://diohabara.github.io/bokuzushi/

## ゲームシステム

- **色の強さ**: 赤 < 橙 < 黄 < 緑 < 青 < 藍 < 紫 < 黒
- **貫通**: ボールの色 > ブロックの色 → 跳ね返らず貫通破壊
- **同色**: ボールの色 = ブロックの色 → 通常破壊（跳ね返る）
- **弱い**: ボールの色 < ブロックの色 → 削りダメージ（跳ね返る）
- **虹色ブロック**: 超硬ブロック（HP20）。削って壊せる
- **段位**: ブロックを壊すとヒット蓄積 → レベルアップでボールの色が昇格
- **星破壊ボーナス**: パドル反射回数が少ないほど高得点

## ワールド

| ワールド | テーマ | ブロック色範囲 |
|----------|--------|--------------|
| 桜花 | 春 | 赤〜橙 |
| 炎祭 | 夏 | 赤〜黄 |
| 紅葉 | 秋 | 赤〜青 |
| 氷雪 | 冬 | 赤〜藍 |
| 黄金 | 極 | 赤〜紫 |

前のワールドをクリアすると次が解放される。
