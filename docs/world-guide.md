# BOKUZUSHI 世界観ガイド

最終確認日: 2026-03-08

## 0. この文書の役目

この文書は、BOKUZUSHI の世界観を決める文書である。  
設定資料ではない。飾りでもない。  
README、UI、演出文言、将来のタイトル変更を判断する時の基準である。

先に結論を書く。

この作品は、**和風パチンコ宇宙を渡る遊技譚** である。  
祭りの熱、夜のざわり、当たりが近い時の湿った気配、それらが宇宙になった場所を進む。  
銀珠を返し、障り札を割り、当たり星へ通す。  
それがこの作品の芯である。

## 1. これは何か

これは冒険譚ではない。  
戦記でもない。  
SF の皮をかぶった説明ゲームでもない。

これは、**盤に座ること自体が旅になる世界** である。

プレイヤーは勇者ではなく、**打ち手** である。  
打ち手は世界を救わない。  
ただ、銀河の奥に沈んだ当たり星まで銀珠を通そうとする。

目的は単純でよい。

- 障り札を割る
- 流れをつなぐ
- ざわりを満たす
- アガりへ入る
- 当たり星へ届く

この順番が、そのままゲームの呼吸になる。

## 2. 世界の事実

この世界には、まず事実がある。  
比喩ではなく、事実としてそうである。

- 宇宙は無数の **銀河** でできている
- 各銀河は桜花・炎祭・紅葉・氷雪・黄金の名を持つ
- 銀河の奥には複数の **巡目** がある
- 巡目が深くなるほど、熱も圧も濃くなる
- 盤面を埋める四角い塊は **障り札** である
- 障り札は流れを止める
- 打ち手は **銀珠** を返して流れを作る
- 銀珠を返す板は **返し板** である
- 各巡目の奥には **当たり星** が沈んでいる
- ざわりが満ちると、盤は **アガり** に入る

ここで大事なのは、全部を説明しないことである。  
意味を語りすぎない。  
だが、輪郭はあいまいにしない。

銀河は銀河である。  
巡目は巡目である。  
当たり星は当たり星である。  
言い切る。濁さない。そこから妙さを立ち上げる。

## 3. この作品の気配

優先するのは、次の感覚である。

- きらびやか
- 祭りめいている
- うさんくさいのに神々しい
- 濡れた熱がある
- もう来る気がする
- 縁起と不穏が同時にある

避けるのは、次の感覚である。

- 乾いたメカ SF
- 漢字だけで固めた観念文
- ネット文化の直輸入
- 業界スラングの貼り付け
- ただの悪ふざけに見える言い回し

和風である。  
だが、古典そのものではない。  
パチンコである。  
だが、現実のホール用語をそのまま並べるだけではない。

**祭りの気配を持った、宇宙の遊技語** にする。

## 4. 言葉の作り方

この作品の言葉は、三つの層で作る。

1. 骨組みは漢字で置く  
   世界の輪郭、役割、目的は漢字で固定する。

2. 熱と湿りはひらがなでにじませる  
   ざわり、ゆらぎ、にじみ、まつり、まだ、もう、など。

3. 状態変化と跳ねはカタカナで鳴らす  
   アガり、ギラつく、ギン、ドクン、など。

つまり、**漢字で締める。ひらがなで湿らせる。カタカナで跳ねさせる。**

この順番を崩すと、ただの難語集になる。  
あるいは、ただのふざけた当て字集になる。  
それは避ける。

## 5. 文体ルール

- 一文は短くする
- 先に断定し、その後で補う
- 反復を恐れない
- 余計な比喩を足さない
- 英語は原則使わない
- 造語は、見ればなんとなく読める形にする
- 説明しすぎないが、意味は逃がさない

目指す文のかたちは、こうである。

- `島を選べ`
- `障り札を割れ`
- `当たり星まで通せ`
- `ざわりが満ちる`
- `アガり突入`
- `飲まれた`

短い。  
強い。  
少し妙で、しかし読める。  
この線を保つ。

## 6. 用語の正規形

| 概念 | 正規語 | 補足 |
| --- | --- | --- |
| プレイヤー | 打ち手 | 主人公というより、盤へ座る者 |
| ワールド | 銀河 | プレイヤー向け表示でも「銀河」を使う |
| ウェーブ / ステージ | 巡目 | `1巡目`, `2巡目`, `3巡目` |
| ブロック | 障り札 | 流れをせき止める札 |
| 目標物 | 当たり星 | 実装上の `Star` は当面そのままでよい |
| ボール | 銀珠 | 打ち手が通す珠 |
| パドル | 返し板 | 銀珠を返す板 |
| レベル | 段位 | 打ち手の乗り具合 |
| 色ティア | 玉色 | UI 表示はこれでよい |
| Fever ゲージ | ざわり | 高まりの予兆 |
| Fever 状態 | アガり | 盤が開いた状態 |
| コンボ | 連なり | 長く続く時は `連チャン` も演出語として可 |
| Game Over | 飲まれた / 流れ切れ | 状況に応じて使い分ける |
| Clear | 銀河抜け / 当たり星到達 | 進行状況で使い分ける |

ただし、現実のパチンコ広告のように並べない。  
文の中で異物感なく光る時だけ使う。

## 8. 銀河の見せ方

現行の銀河名は維持する。

- 桜花
- 炎祭
- 紅葉
- 氷雪
- 黄金

これらは季節ステージではない。  
**銀河の銘** である。

表示の基本形は次の通り。

- `桜花 1巡目`
- `炎祭 2巡目`
- `黄金 3巡目`

`第一層` のような硬い言い方は、原則使わない。  
この作品は層ではなく巡目で進む。

## 10. タイトル方針

`BOKUZUSHI` は repo 名として残っていてよい。  
ただし、表示名は別に持たせる前提で考える。

現時点の候補:

- **ざわり銀河**
- **当たり星行**
- **アガり深度**
- **銀珠渡り**

第一候補は **ざわり銀河**。  
銀河という語が、この作品の広がりと遊技感の両方を持てるためである。
