# BOKUZUSHI 世界観ガイド

最終確認日: 2026-03-08

## 目的

- ゲーム内文言の設定と語彙を一貫させるための基準を定める。
- 実装都合で残っている仮語や揺れを、今後どの方向へ寄せるかを明文化する。
- パチンコ語彙の熱量は借りるが、固有名詞と世界の理屈は BOKUZUSHI 独自のものにする。

## 作品の芯

BOKUZUSHI は「盤を攻略して奥核を暴く」遊技儀式である。

プレイヤーは **星砕き** と呼ばれる攻略者であり、各章に張られた多層の障壁を穿ち、最奥に沈む **奥核** へ到達する。  
見た目や熱量は遊技盤の興奮を思わせるが、世界の住人はそれを賭博や機械ではなく、流れと兆しを読む技法として扱っている。

## 世界観の前提

- 舞台は「台」ではなく **章**。章ごとに風土と気配が異なる。
- 盤面は単なる壁ではなく、流れを濁らせる **障壁層** で構成される。
- 奥にある目標物は「星」そのものではなく **奥核**。星砕きは奥核を暴き、盤の封を解く者を指す称号である。
- ボールは **砕珠**。色が深まるほど、より硬い障壁層を貫ける。
- パドルは **受け盤**。砕珠の流れを受け止め、次の角度へ返す。
- Fever 的な高揚状態は **沸界**。盤の流れが一時的に噛み合い、得点と演出が過熱する。

## 用語の正規形

| 概念 | 正規語 | 意味 |
| --- | --- | --- |
| プレイヤー | 星砕き | 盤を攻略して奥核へ至る者 |
| ワールド | 章 | ひとつの盤域。季節や気配を持つ |
| ウェーブ | 層 | 章の中にある段階的な障壁群 |
| ブロック | 障壁 / 障壁層 | 砕珠の進行を妨げる板片 |
| 目標物 | 奥核 | 各層の奥に置かれた封の核 |
| ボール | 砕珠 | 色と位階を持つ破砕の珠 |
| パドル | 受け盤 | 砕珠の流れを返す受け手 |
| レベル | 段位 | 星砕きとしての到達度 |
| 色ティア | 玉色 / 砕色 | 砕珠が持つ破砕の色位 |
| Fever | 沸界 | 流れが噛み合って熱量が跳ねる状態 |
| コンボ | 連環 | 連続破壊で繋がる勢い |

## 文体ルール

- 日本語を主とし、英語は飾りとして短く添える。
- 命令は短く、強く、盤面に集中させる。
- 説明文は「何をすればよいか」が先、「雰囲気」は後。
- 名詞はできるだけ独自語へ寄せるが、読めない当て字は避ける。
- 「確変」「保留」「右打ち」などの直球パチンコ語彙は、常用語としては採用しない。
- ただし「ざわり」「流れ」「気配」「熱い」など、熱量を伝える一般語は使ってよい。

## 使ってよい語感

- 流れ
- 気配
- 沸く
- 穿つ
- 暴く
- 層
- 封
- 核
- 兆し
- 連なる

## 避けたい方向

- 実在の遊技機ルールを直接連想させる語
- SF に寄りすぎた機械用語
- 和風とホール語彙が同一文内で衝突する文
- 意味よりノリが先に立って読めない造語

## 章の扱い

現行の章名は維持してよい。

- 桜花
- 炎祭
- 紅葉
- 氷雪
- 黄金

これらは「季節のステージ」ではなく、盤域ごとの気配名として扱う。  
画面上では「章」を基本単位にし、必要なら副題で風土を補う。

## 今後の文言整理方針

- UI の「ワールド」は原則 **章** に寄せる。
- 「ステージ」「ウェーブ」はプレイヤー向け表示では **層** に寄せる。
- `Star` に由来する実装名は、コード上の互換性を優先して残してよい。
- 実表示では「星」より **奥核** を優先候補とする。
- 現在の仮語や混在語は、`src/gameContent.ts` と `index.html` で段階的に置換する。

## 例

良い例:

- 章を選べ
- 奥核へ至れ
- 沸界が近い
- 連環が続く
- 障壁を穿て

避けたい例:

- 右打ちの顔
- 確変の口
- 保留が育つ
- 当たりの川
